## 🚀 Project Overview

The **Modern Cordel Project** is a high-fidelity web implementation of the poem "Cordel Moderno" by **Milton Duarte**. This project was developed as a technical challenge during the **Curso em Vídeo (HTML5/CSS3)** program, focusing on advanced layout orchestration and immersive visual effects.

The main goal was to create a reading experience where the traditional aesthetics of Cordel literature meet modern web techniques, specifically the **Parallax Scrolling** effect.

## 🛠️ Technical Implementation

* **Parallax Scrolling:** Implemented high-performance background effects using `background-attachment: fixed`, creating a sense of depth as the user scrolls through the poem's verses.
* **Semantic HTML5:** Built with accessibility and SEO in mind, utilizing proper tags to structure the literary content.
* **Responsive Typography:** Ensuring the poem remains legible and visually striking across all screen sizes, from mobile devices to desktop monitors.
* **Advanced CSS Styling:** Mastery of container positioning, white-space management, and high-contrast color palettes to honor the Cordel visual style.

## ✨ Key Features

- [x] **Immersive Backgrounds:** Fixed images that reveal content through "windows" during scrolling.
- [x] **Fluid Layout:** Fully responsive design that adapts to different viewport widths.
- [x] **Zero Dependencies:** Pure HTML and CSS implementation, highlighting core web development skills.
- [x] **Live Preview:** Deployed via GitHub Pages for instant demonstration.
